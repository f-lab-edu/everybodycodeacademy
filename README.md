# everybodycodeacademy
모두의 코드 아카데미라는 코딩, 코딩테스트를 주제로, 학습 및 리뷰 등이 가능한 커뮤니티 기능을 하는 사이트입니다.

***

### 시스템 구성도
- 로드밸런서 1 ea (Nginx, +nGriner)
- 자바 애플리케이션 3 ea (Docker)
- Mysql 1 ea (EC2 Instance)
- Scouter Collecter Server 1 ea
